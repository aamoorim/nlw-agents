# 📌 Sobre o Projeto

O **NLW Agents** é um projeto frontend desenvolvido durante o evento NLW com foco em estruturação de páginas, estilização e interatividade utilizando JavaScript.

---

# ✨ Funcionalidades

✔️ Interface responsiva  
✔️ Interatividade com JavaScript  
✔️ Layout moderno  
✔️ Organização simples e funcional
